1. Install the browser extension.
  2. Pin the browser extension to the toolbar.
  3. Go to an Ionbiz subdomain url (*.ionbiz.com).
  4. View a ticket detail.
  5. Copy the ticket by:
    1. Left-clicking on the extension in the toolbar. Right-click on the extension to update the copy format. A ticket has multiple copy formats.
    2. Using shortcuts to copy multiple ticket formats. Right-click on the extension to configure the shortcuts.
  6. Share the ticket!
