Ionbiz Ticket Sharing

Getting started

  1. Install the browser extension.
  2. Pin the browser extension to the toolbar.
  3. Go to an Ionbiz subdomain url (*.ionbiz.com).
  4. View a ticket detail.
  5. Copy the ticket by:
    1. Left-clicking on the extension in the toolbar. Right-click to update the copy format.
      Click to view the copy formats. Ticket URL: Displays the ticket url. e.g. https://example.ionbiz.com/Issue/Index/1 Ticket hyperlink: Displays the ticket title and URL as a hyperlink e.g. 12345 Ticket name Ticket title: Displays the ticket title. e.g. 12345 Ticket name Ticket title and URL: Displays the ticket title and URL. e.g. 12345 Ticket name (https://example.ionbiz.com/Issue/Index/1)
    2. Using shortcuts.Right-click on the extension to configure shortcuts.
  6. Share the ticket!
